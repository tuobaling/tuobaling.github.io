ISO 27000:2023-名詞解釋-繁中版

ISO/IEC 27000:2018 繁中版 標準術語 ↔ 日常用語 對照表

第 3.1～3.77 節 術語與定義

3.1 訪問控制（Access Control）

3.2 攻擊（Attack）

3.3 稽核（Audit）

註1： 稽核可以是內部稽核（第一方）或外部稽核（第二方或第三方），可以是結合稽核（結合兩個或兩個以上學科）。

3.4 稽核範圍（Audit Scope）

3.5 鑑別（Authentication）

3.6 真實性（Authenticity）

3.7 可用性（Availability）

3.8 基本測度（Base Measure）

3.9 能力（Competence）

3.10 保密性（Confidentiality）

3.11 符合性（Conformity）

3.12 後果（Consequence）

註1： 一個事態可能導致一系列後果。

3.13 持續改進（Continual Improvement）

3.14 控制措施（Control）

註： 控制措施可能不一定總是達到預期或假定的風險改變效果。 |

3.15 控制目標（Control Objective）

3.16 糾正（Correction）

3.17 整改措施（Corrective Action）

3.18 導出測度（Derived Measure）

3.19 成文資訊（Documented Information）

3.20 有效性（Effectiveness）

3.21 事態（Event）

註1： 一個事態可能是一或多個發生，並可能有多種原因。

3.22 外部脈絡（External Context）

3.23 資訊安全治理（Governance of Information Security）

3.24 治理單位（Governing Body）

3.25 指標（Indicator）

3.26 資訊需求（Information Need）

3.27 資訊處理設施（Information Processing Facilities）

3.28 資訊安全（Information Security）

3.29 資訊安全持續性（Information Security Continuity）

3.30 資訊安全異常（Information Security Event）

3.31 資訊安全事故（Information Security Incident）
